Starwake Protocol — Mobile HUD cache-bust fix

Root cause addressed:
- Every build reused styles.css/game.js filenames.
- Mobile browsers and deployments could continue serving older cached HUD rules.
- This produced the persistent left-side minimap and top-right desktop health bar even after source fixes.

Changes:
- Renamed CSS and JavaScript assets with a unique build suffix.
- Added no-cache document headers.
- Added one final authoritative mobile HUD rule block.
- Fully suppresses the desktop health bar on mobile.
- Keeps pause button and the single minimap together in the top-right HUD row.
